Este repositório contém um projeto de Ciência de Dados que consiste avaliar um modelo de Machine Learning que está buscando prever a possibilidade de Churn de uma empresa de software SaaS. Desenvolvido em Python uso os Modelos de 'Lojistic Regression', 'Random Forest', otimização de hiperparâmetros com o 'GridSearch' e o 'XGBoost' , avaliando através das métricas de 'Accuracy', 'Balanced Accuracy', 'F1 Score', 'Recall' , 'Precision' e 'ROC AUC'.
Para cada avaliação plotamos a 'Confusion Matrix' para avaliar possíveis vícios e overfiting.
